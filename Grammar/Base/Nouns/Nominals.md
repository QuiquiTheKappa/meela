The faunish counting system is decimal.

| Meela      | English |
| ---------- | ------- |
| sho        | one     |
| shosho     | two     |
| shea       | three   |
| shoshea    | four    |
| shoshoshea | five    |
| sheshea    | six     |
| vo         | seven   |
| vosho      | eight   |
| voshosho   | nine    |
| voshea     | ten     |
