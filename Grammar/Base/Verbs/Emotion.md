| ----------- | ------- | --------------- |
| happy       | -sha    | aaolasha        |
| sad         | -shu    | aaolashu        |
| euphoric    | -shasah | aaolashasha     |
| sorrowed    | -gru    | aaolagru        |
| angry       | -grugru | aaolagrugru     |
| confused    | -lu     | aaolalu         |
| adoring     | -bi     | aaolabi         |
| melancholic | -druu   | aaoladruu       |
